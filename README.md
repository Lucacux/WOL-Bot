# WOL-Bot

WOL-Bot is a Discord-based infrastructure power management bot designed to orchestrate Wake-on-LAN (WOL) signals and power schedules for local servers and NAS environments directly via an interactive chat interface.

## Key Features

* **Interactive Control Panel:** Monitor real-time power status (Online/Offline) and trigger remote wake-up signals using native Discord buttons.
* **Automated Power Scheduling:** Set dedicated routine intervals to automatically power up or shut down specific machines (e.g., automated uptime windows).
* **Modern Slash Commands:** Built entirely using modern Discord slash commands for a seamless, prompt-driven user experience.
* **Granular Fleet Visibility:** Easily track internal network IPs and operational states of different home lab nodes from a centralized channel.
